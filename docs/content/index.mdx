## Quick Start

<Steps>
### Install the Extension

Install from [Chrome Web Store](#) (coming soon) or [build from source](https://github.com/prosdevlab/auth-header-injector)

### Open Side Panel

Click the extension icon in your Chrome toolbar to open the side panel

### Enable & Add Rule

Toggle the extension on and add your first auth rule with a URL pattern and token

### Start Browsing

Navigate to matching URLs and see real-time request tracking in action
