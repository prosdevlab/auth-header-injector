## GitHub

### Report a Bug

Found a bug? [Open an issue](https://github.com/prosdevlab/auth-header-injector/issues/new) on GitHub.

Please include:
- Chrome version
- Extension version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if relevant

### Ask a Question

Have a question? [Start a discussion](https://github.com/prosdevlab/auth-header-injector/discussions/new).

Good for:
- "How do I...?"
- Feature requests
- General questions
- Sharing tips and tricks

### View Existing Issues

Check [existing issues](https://github.com/prosdevlab/auth-header-injector/issues) - your question might already be answered!

## Response Time

This is an open source project maintained by volunteers:

- **Bug reports**: Usually within 2-3 days
- **Questions**: Usually within 1-2 days  
- **Feature requests**: Varies based on complexity

For urgent security issues, please open a GitHub issue with `[SECURITY]` in the title.

## Professional Support

This is a free, community-driven project. There is no paid support available.
